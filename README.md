# S&S Distribution - Website

Professional website for S&S Distribution, industrial chemical distribution company in Curaçao.

## About

S&S Distribution specializes in:
- Industrial cleaning chemicals
- Sanitation solutions
- Food-grade lubricants (JAX brand)
- Auto care products
- Marine greases

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Futuristic design with neon effects and glass morphism
- Dark/Light theme toggle
- Multi-language support (English, Spanish, Dutch, Papiamentu)
- Animations with Framer Motion
- Responsive design
- Tailwind CSS for styling
- Next.js 14 with App Router

## Structure

- `/` - Home page
- `/about` - About us
- `/products` - Product catalog
- `/industries` - Industries we serve
- `/brands` - Brands we distribute
- `/contact` - Contact information
