# Client Logos

Add client logos to this folder with the following names:

- `chemicals-global.png` - Chemicals Global logo
- `zymeflow.png` - Zymeflow logo
- `membranes.com.png` - Membranes.com logo

## Instructions:

1. Download logos from client websites (with permission)
2. Save as PNG files with transparent backgrounds (recommended)
3. Recommended size: 200-300px width
4. Name files exactly as shown above
5. Uncomment the logo code in `src/app/brands/page.js`

## Where to find logos:

- Chemicals Global: https://chemicalsglobal.com/
- Zymeflow: https://www.zymeflow.com/
- Membranes.com: https://membranes.com/

**Important**: Always get permission before using company logos!
